# My Portfolio - Anlan Yao

This is my personal portfolio website for the IDM Programming module (2025). It showcases my background, projects, and contact information.

## Live Site
[View the live portfolio]https://anlanyao.github.io/IDM-Programming-2025--Anlan-Yao-/index.html

## Pages
- **Home:** Welcome message with a random greeting.
- **About:** Introduction about myself.
- **Projects:** Displays a list of projects with images, descriptions, and links.
- **Contact:** Contact form with basic submission alert.

## Features
- Uses **HTML5, CSS3, and JavaScript**.
- **Object-Oriented JavaScript**: `Project` class used to create project cards dynamically.
- **DOM Manipulation**: JavaScript updates page content based on data.
- **Reusable Functions**: Functions for initializing pages and rendering content.
